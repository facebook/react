# React 中文文档

用于构建用户界面的 JavaScript 库。

## ✨ 特性

- 🧩 组件化
- ⚡ 虚拟 DOM
- 🔄 单向数据流
- 📱 React Native

## 🚀 快速开始

```bash
npx create-react-app my-app
cd my-app
npm start
```

## 📖 示例

```jsx
function App() {
  return <h1>你好，React！</h1>
}
```
